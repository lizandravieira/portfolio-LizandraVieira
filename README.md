# HTML CV — Frontend Assignment

Este projeto consiste na criação de um currículo utilizando **HTML básico**, desenvolvido como parte de uma atividade acadêmica de introdução ao desenvolvimento web.

O objetivo da atividade foi praticar os principais elementos estruturais do HTML, organizando o conteúdo de um currículo em uma página simples.

---

## 📄 Estrutura da página

O currículo foi organizado utilizando uma **tabela principal**, responsável por distribuir o conteúdo na página de forma alinhada.

A página inclui os seguintes elementos:

* Foto pessoal
* Vídeo de apresentação (via iframe)
* Frame extra com conteúdo incorporado
* Links para páginas favoritas
* Lista ordenada de hobbies
* Lista não ordenada de soft skills
* Tabela com histórico profissional

---

## 🧩 Tecnologias utilizadas

* HTML5

Este projeto utiliza apenas **HTML puro**, sem uso de CSS ou JavaScript, conforme solicitado na atividade.
